功能描述
================================================================================
本样例主要涉及keil5工程的建立

说明：
1、Startup文件存放：core_cm3.c、core_cm3.h、stm32f10x.h、system_stm32f10x.c、system_stm32f10x.h
2、Driver文件存放：gpio\adc\dac\flash\i2c\rcc\rtc\tim\usart\等端口的驱动文件
3、Readme文件存放：工程说明文件
4、Output文件存放：输出文件及临时编译文件
5、User文件存放：main.c\pubilc.c\stm32f10x_conf.h\stm32f10x_it.c\stm3210x_it.h
